## M.1 — Предположения

1. Временное рассогласование $\Delta t \leq 3$ мин достаточно мало, чтобы солнечная активность существенно не изменилась.
2. Метки $y_i$ одинаковы для всех частот в триплете (наблюдения в близкие моменты времени).
3. Seed=42 обеспечивает воспроизводимость.
4. 1000 триплетов достаточно для статистической мощности.

## M.2 — Вопросы

Q1: Какова $S_{3000,6000}$ и $S_{3000,12200}$?
Q2: Зависит ли $S$ от класса ($Ok$ vs $Bad$)?
Q3: Падает ли уверенность $p$ с ростом частоты?
Q4: Есть ли корреляция $S$ с $\Delta t$?
Q5: Стабильна ли $S$ по месяцам?
Q6: Превышает ли модель majority baseline?

## M.3 — Статистические методы

- Bootstrap 95% CI (1000 ресэмплингов, seed=42)
- $\chi^2$ test for equality of proportions (H2)
- One-sided z-test against thresholds (H3)
- Paired t-test for confidence (H5)
- Spearman correlation for $\Delta t$ (H6)
- $\chi^2$ homogeneity test (H7)
- Kruskal-Wallis for monthly variation (H8)
- z-test vs majority baseline (H9)

## M.5 — Интерпретация (для автора)

### Главный вывод
Модель **не достигает практического порога** пригодности для кросс-частотного инференса. Согласованность 67.1% (6000 МГц) и 54.2% (12200 МГц) значительно ниже порога 90%.

### Асимметрия по классам
Модель надёжно распознаёт **Bad**-изображения на всех частотах (79.8–91.9%), но плохо переносит **Ok**-предсказания (22.6–56.4%). Физическая интерпретация: плохие изображения (шум, артефакты) выглядят плохо на любой частоте. Хорошие изображения содержат частотно-зависимую структуру.

### Рекомендации
1. Для **отбраковки данных** (detect bad images) модель можно использовать на всех частотах без дообучения.
2. Для **научного анализа** (нужны надёжные good-метки) требуется дообучение на каждой частоте.
3. Сезонная вариация (36.7–100%) существенна — необходима стратификация по месяцам.

### Ограничения
- $\Delta t \leq 3$ мин — не учтено время суток
- Только 2025 год — нужны дополнительные годы
- Только бинарная классификация
