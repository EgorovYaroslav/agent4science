# MOTIVATION.md

## Исследовательский вопрос

**Насколько хорошо модель классификации радиоизображений СРГ, обученная на данных 3000 МГц, обобщается на данные других частотных каналов (6000 МГц и 12960 МГц)?**

Формально: пусть $f_{3000}$ — модель классификации, обученная на множестве радиоизображений $\mathcal{D}_{3000} = \{(X_i^{(3000)}, y_i)\}_{i=1}^N$, где $X_i^{(3000)}$ — радиоизображение на частоте 3000 МГц, а $y_i \in \{1, \ldots, K\}$ — метка класса (тип солнечной активности).

Рассмотрим два дополнительных множества данных, полученных в близкие моменты времени:
- $\mathcal{D}_{6000} = \{(X_i^{(6000)}, y_i)\}$ — изображения на частоте 6000 МГц,
- $\mathcal{D}_{12960} = \{(X_i^{(12960)}, y_i)\}$ — изображения на частоте 12960 МГц,

при условии $\left| t_i^{(3000)} - t_i^{(6000)} \right| \le 3\ \text{мин}$ и $\left| t_i^{(3000)} - t_i^{(12960)} \right| \le 3\ \text{мин}$.

**Гипотеза:** Если $f_{3000}$ даёт согласованные предсказания на парах $(X_i^{(3000)}, X_i^{(6000)})$ и $(X_i^{(3000)}, X_i^{(12960)})$, то дообучение отдельных моделей на $\mathcal{D}_{6000}$ и $\mathcal{D}_{12960}$ не требуется, что позволяет сэкономить вычислительные ресурсы и время разработки.

**Вопросы, на которые мы отвечаем:**

1. Какова **эмпирическая доля совпадений** предсказаний $f_{3000}$ на парах каналов (3000↔6000 МГц и 3000↔12960 МГц) при $N = 1000$ случайно выбранных временных окнах?
2. Зависит ли качество согласования от **класса солнечной активности**? Есть ли классы, для которых модель «переносится» хуже?
3. Достаточно ли **статистической мощности** эксперимента (1000 испытаний) для принятия решения о пригодности модели к кросс-частотному инференсу?
4. Каковы **практические последствия**: можно ли использовать одну модель для оперативного прогнозирования по всем трём каналам СРГ?

## Форма ответа

Короткий отчёт (`REPORT.md`) с:
- описанием выборки данных (критерии отбора, временные окна, баланс классов),
- таблицей и графиком «доля совпадений по парам каналов» с 95% доверительными интервалами,
- гистограммой распределения согласованности **по классам** (stacked bar plot: канал × класс),
- статистическим тестом (например, $\chi^2$ или точный тест Фишера) на независимость согласованности от частотного канала,
- выводом: «модель применима / не применима для кросс-частотного инференса» с обоснованием,
- приложением: скрипт воспроизведения эксперимента (`experiment.py`) с фиксированным `random_seed` и версией зависимостей (`requirements.txt`).

## Ограничения

- **Данные**: только 2025 год, файлы `.fit` с сервера `ftp.rao.istp.ac.ru`, три канала:
  ```
  3000 МГц:  https://ftp.rao.istp.ac.ru/SRH/SRH0306/cleanMaps/2025MMDD/srh_I_2025-MM-DDThh:mm:ss_3000.fit
  6000 МГц:  https://ftp.rao.istp.ac.ru/SRH/SRH0612/cleanMaps/2025MMDD/srh_2025MMDDThhmmss_6000_I.fit
  12960 МГц: https://ftp.rao.istp.ac.ru/SRH/SRH0306/SRH1224/2025MMDD/srh_2025MMDDThhmmss_12960_I.fit
  ```
  Временное окно совпадения: ≤ 3 минут между снимками разных каналов.
- **Инференс**: только через предоставленный API-эндпоинт `https://forecasting.iszf.irk.ru/api/srh/predict`, без локальной загрузки весов модели.
- **Воспроизводимость**:
  - фиксированный `random_seed = 42` для выборки временных окон,
  - логирование всех запросов и ответов API в `logs/inference_{timestamp}.jsonl`,
  - указание версий библиотек: `requests`, `numpy`, `pandas`, `matplotlib`.
- **Статистика**:
  - оценка доли совпадений с бутстрэп-доверительными интервалами (1000 ресэмплингов),
  - поправка на множественные сравнения при анализе по классам (Бонферрони или Бенджамини–Хохберг).

## Критерии успеха

Эксперимент считается завершённым и информативным, если:

- [ ] Собрано и проверено ≥ 1000 троек файлов (3000/6000/12960 МГц) с временным рассогласованием ≤ 3 мин.
- [ ] Доля успешных ответов API ≥ 95% (остальные — с повторной попыткой или исключением с логированием причины).
- [ ] Отчёт содержит визуализации, интерпретируемые без привлечения исходного кода.
- [ ] Выводы сформулированы в терминах задачи: «модель можно/нельзя использовать для кросс-частотного инференса», с указанием ограничений.
- [ ] Код эксперимента проходит `python -m py_compile experiment.py` и может быть запущен командой:
  ```bash
  python experiment.py --seed 42 --n-samples 1000 --output results/
  ```
